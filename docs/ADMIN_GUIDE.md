# Admin Guide

## الأقسام

1. لوحة القيادة: إحصائيات وطلبات حديثة.
2. الطلبات: بحث وفلترة ومراجعة.
3. الأسئلة: إضافة وتعديل وحذف الأسئلة.
4. القوانين: إدارة مكتبة القوانين.
5. الإعدادات: فتح وغلق التقديم والفصائل.
6. الحظر: Blacklist حسب Client ID أو Discord أو RP Name أو IP Hash أو Device Hash.
7. السجل: آخر تحركات الأدمن.

## نظام المراجعة

لا يوجد نظام نقاط. يتم تقييم الطلب يدويًا عبر:

- الحالة
- الأولوية
- التقييم النوعي
- ملخص المرشح
- سبب القرار
- ملاحظات داخلية
- Flags تلقائية
