# Admin Guide

ادخل إلى `/admin` باستخدام `ADMIN_PASSWORD` الموجود في Environment Variables.

## أقسام اللوحة

- لوحة القيادة: إحصائيات وملخص آخر الطلبات.
- الطلبات: بحث وفلاتر ومراجعة وتغيير حالة.
- الأسئلة: إضافة وتعديل وحذف وتفعيل وتعطيل.
- القوانين: إضافة وتعديل وحذف وتفعيل وتعطيل.
- الإعدادات: فتح/غلق التقديم وتعديل نصوص الموقع والفصائل.
- القائمة السوداء: حظر Client ID أو Discord أو RP Name أو IP Hash أو Device Hash.
- سجل النشاط: متابعة حركات الأدمن.

## نظام القرار

لا يوجد نظام نقاط رقمي. يستخدم الأدمن تقييمًا نوعيًا:

- مرشح قوي جدًا
- مرشح جيد
- يحتاج مراجعة
- غير واضح
- غير مناسب

وحالات طلب:

- جديد
- قيد المراجعة
- يحتاج مقابلة
- مقبول مبدئيًا
- مقبول نهائيًا
- مرفوض
- مؤجل
- مكرر
- محظور
